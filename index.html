JavaScript Tasks (If Else & Comparison Operators)

Task 1: Check if User is Adult or Minor

Ask the user for their age. If the age is 18 or more, show: "You are an adult." Else show: "You are a minor."

Task 2: Positive / Negative Number Check

Ask the user to enter a number. If it's positive, show: "Positive number"; else show: "Not positive".

Task 3: Check Odd or Even

Ask the user for a number. If it's even, alert: "Even number"; else alert: "Odd number".

Task 4: Admin Login

Ask for a username. If the username is "admin", display: "Welcome, admin!". Else display: "Access denied."

Task 5: Compare Two Numbers (Equality)

Ask for two numbers. If they are equal, show: "Both numbers are the same"; otherwise, show: "The numbers are different".

Task 6: Compare Which Number is Bigger

Ask the user to enter two numbers. Check which one is larger and alert accordingly.

Task 7: Number Range Check (0–100)

Ask the user for a number. If it's greater than 100, alert "Too big!"; else "Within range".

Task 8: Student Score Pass/Fail

Ask for a score. If it’s 50 or higher, alert "You passed!"; else "You failed".

Task 9: Temperature Hot/Not

Ask for a temperature. If over 30, alert "It's hot!"; otherwise "It's not hot".

Task 10: Zero or Not Zero

Ask the user to enter a number. If it's 0, alert "Zero"; else alert "Not zero".

Task 11: Check if Senior Citizen

Ask the user for their age. If it's 60 or more, alert "Senior citizen"; otherwise "Not a senior".

Task 12: Are You Faiz Ilham?

Ask the user for their name. If it's "Faiz Ilham", greet them; otherwise, say they aren't Faiz Ilham.

Task 13: Check Password

Ask the user for a password. If it matches "12345", alert "Correct password"; else "Wrong password".

Task 14: Greet Based on Time

Ask for current hour (0–23). If less than 12, alert "Good morning"; otherwise "Good afternoon".

Task 15: Divisible by 5

Ask the user to enter a number. If divisible by 5, alert "Divisible by 5"; else "Not divisible by 5".

Task 16: Want to Play a Game?

Ask the user if they want to play. If they say "yes", alert "Let's play!"; otherwise "Maybe next time!".

Task 17: Favorite Color

Ask the user for their favorite color. If it's "blue", alert "Cool choice!"; otherwise, "Interesting color!".

Task 18: Check Day (Weekend or Not)

Ask for the day of the week. If it's "Sunday", alert "Weekend!"; else alert "Weekday".

Task 19: Fruit Availability

Ask the user to enter a fruit. If it's "banana" or "apple", alert "We have that!"; else "Out of stock".

Task 20: Start the Game

Ask the user to type "start". If they do, alert "Game started!"; else alert "Waiting for input...".

Task 21: Age-Based Grade Check

Enter student marks. Get Distinction, Pass, Fail or Invalid based on score (0–100).

Task 22: Discount Eligibility

Check if the user qualifies for Senior or Premium discounts based on age and membership.

Task 23: Email Verification

Determine login access based on email and verification status.

Task 24: Contest Eligibility

Check if user (18–35, Pakistan/India, not banned) can join the contest.

Task 25: Subscription & Login

Check if user is Premium, Free, or not logged in.

Task 26: Product Purchase

Determine product purchase options based on price, stock, and membership.

Task 27: Work Hours Check

Identify if it’s overtime, weekend work, regular, or no work.

Task 28: Age & Gender Classification

Check if the user is a Teen Male, Teen Female, Adult, or Child.

Task 29: Weather Warning System

Get alerts based on temperature and humidity (Heatwave, Cool, etc.).

Task 30: Score Grading System

Evaluate score (0–100) and return Excellent, Good, Average, etc.

Task 31: Course Access Control

Check user access based on login, enrollment, and ban status.

Task 32: Driving Conditions

Check driving behavior based on speed and license.

Task 33: Order Processing Status

Determine order status based on payment and shipping info.

Task 34: Battery Status Checker

Evaluate battery level and charging condition.

Task 35: Browser & Device Compatibility

Check browser and device for feature support.
